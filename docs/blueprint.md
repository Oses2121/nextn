# **App Name**: BizSight

## Core Features:

- Dashboard Overview: Display a dashboard with key metrics such as total revenue, expenses, and profit, and date ranges for all the displayed info.
- Revenue Tracking: Input and categorize revenue transactions with the source, date, and amount. Have options to view data as yearly, quarterly, or monthly.
- Expense Tracking: Input and categorize expenses with vendor, date, and amount. Be able to create customizable expense reports.
- Profit Calculation: Automatically calculate profit based on revenue and expenses for a desired time period, as well as export capabilities.
- Appointment Calendar: A simple calendar to schedule and view appointments.

## Style Guidelines:

- Primary color: Moderate blue (#6699CC) for a professional and trustworthy feel.
- Background color: Light gray (#F0F0F0) to provide a clean, neutral backdrop.
- Accent color: Soft green (#8FBC8F) to highlight key metrics and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif that is both modern and approachable.
- Use clean and simple icons for each feature, such as a dollar sign for revenue, a graph for profit, and a calendar for appointments.
- Dashboard should be well-organized and intuitive, with key metrics prominently displayed at the top.
- Use subtle transitions and animations when navigating between sections to enhance user experience.
